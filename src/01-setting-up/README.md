# Setting up the Project

## Prerequisites
First you'll need Node.js and the package manager
- Check for existing installation:
```bash
node -v
```
- If Node.js is not installed, go to the official Node.js website and download: https://nodejs.org/

## Installation and Setup
- Clone the Repository
```bash
git clone https://github.com/ngothilinh20187179/react-tutorial.git
```
- Navigate to the Project Directory
```bash
cd react-tutorial
```
- Install Dependencies
```bash
npm install
```
- Running the Application
```bash
npm run dev
```
<!-- Open your web browser and navigate to the address displayed in the terminal (usually http://localhost:5173/). -->

## Creating a New React Project with Vite
- Create the Project
```bash
npm create vite@latest
```
+ Project name: e.g. react-tutorial
+ Select a framework: React
+ Select a variant: JavaScript/TypeScript
- Install and Run
```bash
cd react-tutorial
npm install
npm run dev
```
